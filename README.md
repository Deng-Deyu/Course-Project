# 课程项目

一个基于 Python 的机器学习项目，用于数据分析、模型开发和交互式可视化。

## 项目概述

本项目实现了一个完整的机器学习流程，包括数据处理、模型训练和交互式可视化功能。

## 项目结构

```
├── src/              # 源代码和主要脚本
├── data/             # 输入数据目录
├── outputs/          # 生成的输出和结果
├── report/           # 项目报告和文档
├── requirements.txt  # Python 依赖
└── README.md         # 本文件
```

## 技术栈

- **数据处理**: Pandas, NumPy
- **机器学习**: Scikit-learn, LightGBM, PyTorch
- **模型解释**: SHAP
- **数据可视化**: Matplotlib, Seaborn, Plotly
- **Web 界面**: Streamlit

## 安装

1. 克隆仓库：
```bash
git clone https://github.com/Deng-Deyu/Course-Project.git
cd Course-Project
```

2. 安装依赖：
```bash
pip install -r requirements.txt
```

## 环境要求

- Python 3.8 或更高版本
- 详见 `requirements.txt` 中的具体包版本

## 使用方法

安装完成后，可以运行 Streamlit 应用：
```bash
streamlit run src/app.py
```

## 目录说明

- **src/**: 包含所有源代码和主应用程序文件
- **data/**: 放置输入数据集的目录
- **outputs/**: 存储生成的结果、模型和预测结果
- **report/**: 包含分析报告和文档
